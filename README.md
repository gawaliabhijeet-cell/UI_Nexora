# 🛍️ Nexora - Electronics E-commerce Website

A modern and responsive **Electronics E-commerce Website** built using **React**, **Vite**, and **Tailwind CSS**. The project features a clean UI with a professional footer.

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/gawaliabhijeet-cell/UI_Nexora.git
```
### install vite
```bash
npm create vite
```
### Go to the project folder

```bash
cd UI_Nexora
```

### Install dependencies

```bash
npm install react-icons
npm install lucide-react
```

### Start the development server

```bash
npm run dev
```
